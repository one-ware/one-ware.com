## VHDL Project Guide
[This guide](/docs/one-ai/guides/vhdl-guide) showcases how to run a OneAI model on an FPGA for a demo use case. The use case will be the classification of handwritten digits. It demonstrates how to test AI on any FPGA, regardless of manufacturer and type, by sending images via UART and receiving results back. You can identify your own handwritten digits using an FPGA board of your choosing and the webcam of your computer.
